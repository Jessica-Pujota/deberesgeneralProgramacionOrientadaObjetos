# Sistema de Gestión de Automóviles

## Descripción
Este proyecto es una aplicación básica desarrollada en Python que permite gestionar automóviles utilizando Programación Orientada a Objetos (POO).

## Conceptos de POO Aplicados
- **Herencia:** La clase AutoElectrico hereda de la clase Automovil.
- **Encapsulación:** Uso de atributos protegidos y métodos getters.
- **Polimorfismo:** El método descripcion() se comporta de manera diferente según el tipo de automóvil.
- **Modularidad:** El proyecto se organiza en carpetas de modelos, servicios y un archivo main.

## Estructura del Proyecto
- `modelos/`: Contiene las clases del sistema.
- `servicios/`: Contiene la lógica del sistema.
- `main.py`: Punto de entrada de la aplicación.

## Ejecución
Ejecutar el archivo `main.py` desde el IDE o la terminal.
