# Sistema de Gestión de Citas Médicas

## Descripción
Este proyecto implementa un sistema básico de gestión de citas médicas utilizando Programación Orientada a Objetos (POO) y modularidad en Python.

## Estructura
- main.py: inicia el sistema.
- modelos: contiene las clases Medico y Paciente.
- servicios: contiene la lógica de la clínica.
- README.md: documentación del proyecto.

## Funcionalidades
- Visualizar médicos disponibles.
- Agendar citas médicas.
- Control de disponibilidad.
